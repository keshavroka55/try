A **Gamified Task Management web application** inspired by *Solo Leveling*.  
Users complete tasks, gain XP, level up, track daily consistency, and compete on a **leaderboard** to see the top players.

---

## Features

- Add, edit, and delete tasks  
- **Level Up System**: Gain XP for completing tasks and level up  
- **Consistency Tracking**: Daily streak tracking  
- **Leaderboard**: Shows top players based on XP and consistency  
- Gamified UI inspired by Solo Leveling  
- Simple and responsive interface

## Tech Stack

- **Backend**: Django  
- **Database**: SQLite  
- **Templates**: Django Templates  
- **Styling**: CSS / Bootstrap  
- **Authentication**: Django Auth + Google OAuth (`django-allauth`)  
- **Deployment**: Gunicorn + Nginx on Linux server
